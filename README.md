# SurrealDB Query Tool

<img src="https://repository-images.githubusercontent.com/436658287/48975688-cf92-4b36-9fe1-fbe0a492a74b" alt="SurrealDB">

This project is made using **QT** and **Pyside6**.

## Description

It uses the surrealDB python sdk to query the database

## Installation

1. Clone the repository.
2. Make a virtual environment.
3. Install the necessary dependencies (QT6 and Pyside6).
4. Install surrealDB python sdk.
5. Run the project.
